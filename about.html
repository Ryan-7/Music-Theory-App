About this App

As a guitar player, one of the most frustrating things is having to constantly look up various keys, notes and backing tracks on YouTube. This app aims to help solve that problem by having all of those resources in one place. The second problem this web app aims to solve is helping users figure out which notes will sound best over any of the given 12 Major Keys. Those who need guidance in the right direction pertaining to musical improvisation will especially find this useful.

This is an ongoing project that will be continually updated with new features to help guitar (and other musical instrument) players.

Coming features:

  • Multiple backing tracks per key
  • Drag and drop chord progression creator
  • More complex common progressions
  • Minor Keys

This is an ongoing project - check back frequently for new features. Cheers!
